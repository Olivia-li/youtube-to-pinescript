```
//@version=4
strategy("UTBot Alerts Strategy", overlay=true)

// UTBot Alerts Indicator
utbotAlerts = input(title="UTBot Alerts Settings", type=input.string, defval="1;0.003;0.0015;0.00075")
[utbotLength, utbotFactor1, utbotFactor2, utbotFactor3] = str.split(utbotAlerts, ";")
utbot = security(syminfo.tickerid, timeframe.period, utbot(utbotLength, utbotFactor1, utbotFactor2, utbotFactor3))
plot(utbot, color=color.blue, linewidth=2)

// Linear Regression Candles Indicator
linRegCandlesLength = input(title="Linear Regression Candles Length", type=input.integer, defval=20)
linRegCandles = linreg(close, linRegCandlesLength, 0)
plot(linRegCandles, color=color.orange, linewidth=2)

// Moving Average 200 Indicator
ma200 = sma(close, 200)
plot(ma200, color=color.red, linewidth=2)

// Buy and Sell Signals
buySignal = utbot == 1 and close > ma200
sellSignal = utbot == -1 and close < ma200

// Strategy Entry and Exit
if buySignal
    strategy.entry("Buy", strategy.long)
if sellSignal
    strategy.entry("Sell", strategy.short)

// Plot Buy and Sell Arrows
plotshape(buySignal, style=shape.arrowup, color=color.green, size=size.small, location=location.belowbar)
plotshape(sellSignal, style=shape.arrowdown, color=color.red, size=size.small, location=location.abovebar)
```

This script uses the UTBot Alerts, Linear Regression Candles, and Moving Average 200 indicators to identify buy and sell signals. The `utbotAlerts` input allows you to customize the settings of the UTBot Alerts indicator.

The `strategy.entry()` function is used to enter trades when the buy or sell signal is given. The `plotshape()` function is used to plot buy and sell arrows on the chart.
