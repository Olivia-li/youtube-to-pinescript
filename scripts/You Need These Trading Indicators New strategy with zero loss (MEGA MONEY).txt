```
//@version=4
strategy("Trader XO Micro Trend Scanner Strategy", overlay=true)

// Trader XO Micro Trend Scanner
fast_ma = input(title="Fast MA", type=input.integer, defval=5)
slow_ma = input(title="Slow MA", type=input.integer, defval=10)
fast_ma_val = ema(close, fast_ma)
slow_ma_val = ema(close, slow_ma)
long_signal = crossover(fast_ma_val, slow_ma_val)
short_signal = crossunder(fast_ma_val, slow_ma_val)

// Moving Average Convergence Divergence (MACD)
fast_length = input(title="Fast Length", type=input.integer, defval=12)
slow_length = input(title="Slow Length", type=input.integer, defval=26)
signal_length = input(title="Signal Smoothing", type=input.integer, defval=9)
macd_val = macd(close, fast_length, slow_length, signal_length)
macd_histogram = macd_val - signal(macd_val, signal_length)

// Moving Average Exponential (EMA)
ema_length = input(title="EMA Length", type=input.integer, defval=200)
ema_val = ema(close, ema_length)

// Buy and Sell Signals
buy_signal = long_signal and close > max(fast_ma_val, slow_ma_val) and macd_histogram > 0 and close > ema_val
sell_signal = short_signal and close < min(fast_ma_val, slow_ma_val) and macd_histogram < 0 and close < ema_val

// Strategy Execution
if (buy_signal)
    strategy.entry("Buy", strategy.long)
if (sell_signal)
    strategy.entry("Sell", strategy.short)
```

It also uses the `strategy.entry()` function to execute trades based on the signals generated by the indicators. You can adjust the input parameters to customize the script to your liking.
